## Install (free starter)

### In Cowork (Claude Desktop App)
1. Go to **Plugins**
2. Add via GitHub: `lukasglarborg/student-kit`
3. Say **"hello"** — the kit sets up your workspace in 2 minutes

### In Claude Code (Terminal)
```bash
claude --plugin-dir path/to/student-kit
```

## How to use it

Just talk like you would to a smart classmate:

| You say | What happens |
|---------|-------------|
| "Good morning" | Daily briefing with your day's overview |
| "Make flashcards from today's lecture" | Q&A cards with spaced repetition |
| "Note: pick up the textbook tomorrow" | Saved instantly to your inbox |
| "What were we working on?" | Summary of recent sessions |

**Works in any language.** English, Danish, German, French, Spanish — just answer in your language during setup.

## Academic integrity

Student Kit helps you **learn better**, not cheat. It will never write your assignments. It explains concepts, generates practice material, organizes your notes, and gives feedback on your writing — like a good tutor would.
